#### TODO:
* Everything (WIP)

#### Requirements:
* The source code
* An Arduino Uno
* MG90S 9G Metal Gear Servo Motor
* Adafruit SSD1306 Arduino Library (Display)


#### Instructions:
###### Display Wiring:
* VCC to 3.3 V
* GND to GND
* SDA to A4
* SCL to A5

###### Breadboard Wiring:
* Power to Power
* Ground to Ground
* SDA to A4 Same Row, (Placement does not matter as long as they are on the same row in the same divided area)
* SCL to A5 Same Row, (Placement does not matter as long as they are on the same row in the same divided area)
